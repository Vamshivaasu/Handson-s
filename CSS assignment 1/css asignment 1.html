What is CSS?

  1. CSS stands for Cascading Style Sheet.
  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  3. CSS not neccesaryCSS saves a lot of work. It can control the layout of muliple web pages all at one
  4. External stylesheets are stored in CSS file

CSS can be added to HTML documents in 3 ways:

  • Inline- by using style attribute inside the HTML elements.
  • Internal- by using a style element in the head section.
  • External- by using link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS files.However,in this tutorial we will use inline and Internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

when comparing CSS class vs ID, the difference is that CSS class applies to multiple elements. ID, on other hand, applies to style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS. selectors are used to target a specific element on a web page. Once an element ha sbeen targeted, a style can be applied to the element.

There is a wide range of selectors availabe. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
